# Терминалын нийтлэг командууд
## Терминалтай ажиллах

	ctrl+alt+t 
товчийг дарж terminal нээнэ. 

Терминал дээр ажилладаг нийтлэг коммандууд:

 ls - хавтас доторхыг жагсаана
 cd - хавтас солино

Хавтас руу орох

	cd  folder_name 
	cd /home/zoloo/Desktop

Хавтсаас гарах

	cd ..

Хавтас үүсгэх
	
	mkdir folder_name

	
pwd - Одоо байгаа хавтасын замыг харах

rm - Файл устгах

	rm hello.cpp

rm -R -Хавтас устгах

	rm -R folder
